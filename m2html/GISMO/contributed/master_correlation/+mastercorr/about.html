about

PURPOSE

Tools for master event cross correlation against long time series.

SYNOPSIS

function about

DESCRIPTION 

Tools for master event cross correlation against long time series.
 These tools are designed to run one or more short waveform "snippets"
 against long time series looking for portions of the continuous record
 that correlate above a given threshold. Additional tools help interact
 and extract the results. See the cookbook for a full example.
