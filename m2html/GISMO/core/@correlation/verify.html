verify

PURPOSE

VERIFY

SYNOPSIS

function c = verify(c)

DESCRIPTION 

VERIFY
   C = VERIFY(C) checks the length of waveforms in a correlation object
   and adjusts them if they are not equal. Traces ends are trimmed or
   zero-padded to the mode length of all traces.
