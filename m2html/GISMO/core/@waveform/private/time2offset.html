time2offset

PURPOSE

time2offset returns the offset within a waveform's data for time T

SYNOPSIS

function [offset actualTime] = time2offset(w,whichTime)

DESCRIPTION 

time2offset returns the offset within a waveform's data for time T
 If the time is prior to existing samples, then offset=0 and actualTime=[]
