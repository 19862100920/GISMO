load_irisdmcws

PURPOSE

LOAD_IRISDMCWS loads waveforms using the IRIS Web Services Java Library

SYNOPSIS

function wavef = load_irisdmcws(request)

DESCRIPTION 

 LOAD_IRISDMCWS loads waveforms using the IRIS Web Services Java Library
 For more information about the IRIS Web Services Library for Java,
 check out

 http://www.iris.edu/manuals/javawslibrary/

 See also javaaddpath waveform
 request.combineWaves is ignored

0016    
0017    datefmt = @(dt) datestr(dt, 'yyyy-mm-dd HH:MM:SS.FFF');
0018    idx = 0;
0019    for d = 1 : numel(sTime);
0020        for tag = allChanInfo
0021           thisWave = irisFetchTraces(...
0022              tag.network, tag.station, tag.location, tag.channel, datefmt(sTime(d)), datefmt(eTime(d)));
0023           nWaves = numel(thisWave);
0024           if nWaves > 0
0025              wavef(idx + 1 : idx + nWaves) = thisWave;
0026              idx = numel(wavef);
0027           end;
0028        end
0029    end
0030    wavef = addhistory(clearhistory(wavef),'Imported from IRIS DMC');
