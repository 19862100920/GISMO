traceToWaveform

traceToWaveform converts traceobjects to waveforms

DESCRIPTION 

traceToWaveform converts traceobjects to waveforms

 Note: this may return multiple waveform objects, depending upon

0024    % 1st loop: find signal spikes, and get header info.
0025    for seg = 1:traceCount
0026       obj.trpointer.record = seg - 1;
0027       allw(seg) = waveformMetadata(wBlank, obj.trpointer);
0028       spikes(seg).mask = findSpikes(obj.trpointer);
0029    end
0030    
0031    allw = applyCalibrationsToAll(trpointer, allw);
0032    
0033    % 2nd loop: assign data to the waveforms.
0034    replaceWithNan = @(W,BAD) setsamples(W, BAD.mask, nan);
0035    for seg = 1:traceCount
0036       obj.trpointer.record = seg - 1;
0037       allw(seg) = set(allw(seg), 'data', trextract_data(obj.trpointer));
0038       allw(seg) = replaceWithNan(allw(seg), spikes(seg));
0039    end
