Creating a filterobject


f = filterobject(type, cutoff, poles)
creates a filter from consituent parts. valid values for type are 'b' for bandpass, 'L', for low-pass, and 'H' for high-pass. The cutoff is a single value for either low-pass or high-pass filters, or 2-dimensional vector in the case of a bandpass filter.

  1. f = filterobject('l',0.02, 2)
    creates a low-pass filter for 50 second and longer signals with two poles.
  2. f = filterobject('b',[.8 5], 3 )
    creates a band-pass filter for the standard range 0.8 to 5 hz, with three poles
  3. f = filterobject('h',3,2)
    create a high-pass filter for 3 hz and higher signals.

f = filterobject
create a default filterobject. This is a band-pass filter for between 0.8 and 5 hz, with two poles.


f = filterobject(somefilter)
makes a duplicate filterobject from another one. This is a behind the scenes sort of behavior, and for all intents and purposes you would just use newfilt = oldfilt.

What's in a filterobject?

A filterobject contains only the following information:

  • type
  • cutoff
  • poles

You can use the struct(filterobject) command to return the absolutely raw data and see for yourself... struct returns a structure based on the listed object.

However, much more information can be asked of it via the filterobject/get routine.

  • get(w,'type') - returns the type of filter, either 'B', 'L', or 'H' -- corresponding to Bandpass, Low-Pass, and High-pass respectively.
  • get(w,'cutoff') - returns the frequency cutoff.
  • get(w,'poles') - returns the number of poles used in this filter

Making direct changes to your filterobject

Using set, you can change the values of your filter data. The general form of set is:

modifiedFilterobject = set(originalFilterobject, 'PropertyToChange', newValue)

  • set(w,'type',value) - valid values for type are 'b' for bandpass, 'L', for low-pass, and 'H' for high-pass.
  • set(w,'cutoff',value) - The cutoff is a single value for either low-pass or high-pass filters, or 2-dimensional vector in the case of a bandpass filter. This must be less than the nyquist frequency of your waveform.
  • set(w,'poles', value) - set the number of poles to be used in this filter.
