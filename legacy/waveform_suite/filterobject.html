The Filterobject class

Filterobjects provide a simple way to apply filters to waveforms.

The filterobject class is an extremely simplistic class that is capable of applying a zero-phase butterworth filter to a waveform object. The advantage of this class as opposed to matlab's built-in filtfilt is that it knows how to interpret waveform data (including knowledge of nyquist frequencies & such), updates a waveform's history with the filter that was applied, and returns a waveform. Whereas, all the book keeping for filtering would otherwise have to be handled by the programmer.

Filterobjects are simple in that they only require three pieces of information:

  • Filter Typedescribes the behavior of the filter. The type may be high-pass, low-pass, or bandpass
  • Cutoff Value(s) are the frequencies describing the edges of the filter. For high-pass filters, it is the lowest frequency to pass. For low-pass filters, it dictates the highest frequency allowed to pass. For bandpass filters, this is a pair of numbers, representing both ends of the passband.
  • Poles are the number of poles created for the filter

Creating a filterobject

f = filterobject(type, cutoff, poles)
creates a filterobject with the chosen parameters.
  1. f = filterobject('l',0.02, 2)
    creates a low-pass filter for 50 second and longer signals with two poles.
  2. f = filterobject('b',[.8 5], 3)
    creates a band-pass filter for the standard range 0.8 to 5 hz, with three poles
  3. f = filterobject('h',3,2)
    create a high-pass filter for 3 hz and higher signals.
f = filterobject
create a default filterobject. This is a band-pass filter for between 0.8 and 5 hz, with two poles.

Functions of note

filtfilt
apply a zero-phase butterworth filter to the waveform(s). Because this runs both forward and backward through the data, it effectively doubles the filter's poles.
  1. filteredwave = filtfilt(filterobject,waveform)
    filters a waveform
  2. filtereddata = filtfilt(filterobject,data,nyq)
    Use without a waveform
