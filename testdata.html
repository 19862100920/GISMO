What is GISMO test data?

This directory contains test data in different formats. These are used by the testing routines in GISMO to verify that code modifications do not break the code.

  • css3.0: Databases compatible with Antelope
    • antelope2waveform_testdb: wfdisc
    • avodb200903: event origin
    • avodb20080712: event origin
    • demodb: wfdisc
    • rtdb200903: event origin
  • MiniSeed
    • REF.EHZ.2009.079
    • REF.EHZ.2009.080
    • REF.EHZ.2009.081
    • REF.EHZ.2009.082
    • RSO.EHZ.2009.079
    • RSO.EHZ.2009.080
    • RSO.EHZ.2009.081
    • RSO.EHZ.2009.082
  • RSAM
    • MOMN2015.DAT
  • sac
    • example_sacfile.sac
    • IU.COLA.10.BHZ.M.2002.307.220803.SAC
    • REF.EHZ.2009-03-22.sac
  • sacpz
    • SACPZ.IU.COLA.10.BHE
    • SACPZ.IU.COLA.10.BHN
    • SACPZ.IU.COLA.10.BHZ
  • seisan
    Part of the Seisan database for the Montserrat Volcano Observatory event catalog
    • REA/MVOE_/1996/11
      1 month of the MVO event catalog. The directory contains one Sfile per event.
    • WAV/MVOE_/1996/11
      Contains 1 segmented waveform file (in Seisan format) for each Sfile in REA.
