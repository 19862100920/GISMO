isempty

PURPOSE

ISEMPTY returns TRUE if waveform contains no data

SYNOPSIS

function TF = isempty(w)

DESCRIPTION 

ISEMPTY returns TRUE if waveform contains no data
   TF = isempty(waveform);

   if multiple waveforms are passed, this returns 'true' ONLY if ALL
   waveforms are empty.


  See also ISEMPTY
